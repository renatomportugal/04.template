Team Stacked
The main objective of the stacked avatar group:
  • To display a list of avatars, overlapped (left to right) with the avatars laid over each other
  • The avatars should be overlaid on top of each other, so that the first avatar is fully visible
  • The stacked avatars container should be scrollable (horizontally) on both desktop and mobile when needed
