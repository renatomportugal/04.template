 Using addPhysics() adds physics to any display object in ZIM.
ZIM uses the Box2D engine (Angry Birds) with ZIM helpers
to really wrap up physics and make it easy!
See https://zimjs.com/physics/ and https://zimjs.com/docs.html?item=Physics
ZIM provides JavaScript conveniences, components and controls for the Canvas
and is powered by CreateJS with its solid Bitmap Object Model (BOM).
