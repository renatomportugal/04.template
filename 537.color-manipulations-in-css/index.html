Color Maninpulations in CSS

Given the base set of color values for a single color:

:root {
  --h: 100;
  --s: 50%;
  --l: 50%;
  --hsl: var(--h), var(--s), var(--l);
  --hslf: hsl(var(--hsl));
}

You can play with color in any of the following ways.

Alpha Channel

Base color with variations using the alpha channel

background-color: hsla(
  var(--hsl),
  1
);

background-color: hsla(
  var(--hsl),
  .8
);

background-color: hsla(
  var(--hsl),
  .6
);

background-color: hsla(
  var(--hsl),
  .4
);

background-color: hsla(
  var(--hsl),
  .2
);
