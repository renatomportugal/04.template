JQuery Switch

Apply a switch style for checkboxes

 

Enable - Disable - Check - Uncheck

 

Enable - Disable - Check - Uncheck

 

Using selector jquery for all checkboxes

Enable - Disable - Check - Uncheck

 

Create

$('input:checkbox').switchStyle();

// with options
$('input:checkbox').switchStyle({
       label_on: 'ON',
       label_off: 'OFF',
       label_size: 12,
       width: 80,
       height: 30
});

Enable

$('input:checkbox').switchStyle('enable');

Disable

$('input:checkbox').switchStyle('disable');

Check

$('input:checkbox').switchStyle('on');

Uncheck

$('input:checkbox').switchStyle('off');

 

Rules

- Do not include a "label" element for the checkbox. Use the plugin functions instead
- If you want more then one Switch Button inline, change the CSS property display in ".switch-wrapper" class to inline-block
